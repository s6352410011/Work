1. ข้อมูลทั่วไป
1.1 วันที่รับไว้ในโรงพยาบาล เวลา น. มาถึงหอป่วย โดย เดินมา รถนั่ง เปลนอน
ผู้ให้ข้อมูล ผู้ป่วย ญาติ ผู้นำส่ง
1.2 อาการสำคัญที่มาโรงพยาบาล
1.3 ประวัติการเจ็บป่วยปัจจุบัน
1.4 การวินิจฉัยเบื้องต้น
1.5 ประวัติการแพ้ยา หรือสารต่างๆ ไม่มี มี ( ยา อาหาร อื่นๆ ระบุ ) อาหารที่แพ้
2. สภาพร่างกาย
2.1 ระดับความรู้สึกตัว รู้สึกตัวดี ไม่รู้สึกตัว สับสน ซึม
2.2 การหายใจ ปกติ เจ็บเวลาหายใจ หายใจลำบาก อื่นๆ ระบุ
2.3 การไอ ไม่มี มี มีเสมหะ อื่นๆ ระบุ
2.4 การไหลเวียนโลหิต
ต่อมน้ำเหลือง ปกติ คลำพบต่อมน้ำเหลืองผิดปกติบริเวณ สีผิว ปกติ ผิดปกติ ระบุ
อาการบวม ไม่บวม บวมบริเวณ สีผิว ปกติ ผิดปกติ ระบุ
2.5 การติดต่อสื่อสาร : หู ได้ยินชัดเจน ได้ยินไม่ชัดเจน: การใช้อุปกรณ์ช่วยฟัง มี ไม่มี

3. สภาพจิตใจ ค่านิยม ความเชื่อ และอัตมโนทัศน์
3.1 การแสดงออกทางด้านพฤติกรรม ปกติ ไม่ปกติ ระบุ
3.2 การแสดงออกทางอารมณ์และจิตใจ ปกติ วิตกกังวล กลัว ซึมเศร้า สูญเสียภาพลักษณ์
วิธีแก้ไข ปรึกษาผู้ใกล้ชิด ทำงานอดิเรก แยกตัว ใช้ยา/สารเสพติด อื่นๆ
3.3 สิ่งที่กังวลใจของผู้ป่วยขณะพักรักษาในโรงพยาบาล
3.4 บุคคลที่มีความสำคัญและผู้ป่วยต้องการอยู่ใกล้ชิดมากที่สุด บิดา มารดา บุตร เพื่อน บุคคลอื่น ระบุ
3.5 บุคคลที่ติดต่อได้/ผู้ดูแล เบอร์โทรศัพท์ที่สามารถติดต่อได้
3.6 ค่านิยมและความเชื่อ  การเจ็บป่วยครั้งนี้เชื่อว่า อุบัติเหตุ การปฏิบัติตนไม่ถูกต้อง ตามวัย เคราะห์กรรม อื่นๆ
สิ่งยึดเหนี่ยวทางจิตใจ ไม่มี มีระบุ
